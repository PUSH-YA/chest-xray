# chest-xray

<br>

### Conda Environments: 
#### Feature Eng
Create the conda environment: 

`conda env create -f environment.yml`

Activate the conda environment: 

`conda activate xray-analysis`
<br>

#### SVC Processing
Create the conda environment: 

`conda env create -f inference_env.yml`

Activate the conda environment: 

`conda activate xray-svc`
